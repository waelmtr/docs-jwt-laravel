Introduction

A simple REST API in Laravel

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Register

This endpoint is used to register a user to the system.

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"wael\",
    \"email\": \"ian@gmail.com\",
    \"password\": \"12345678\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "wael",
    "email": "ian@gmail.com",
    "password": "12345678"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, Successful Register):


{
"message": "User Successfully Registered",
"user : {
 "name" : "...."
 "email" : "...."
 }
}

Example response (400, Failed Reqister):


{
"field": [ "the Field is required" ]
Example :  "name": [
  "The name field is required."
  ],
}

Logout

requires authentication

this is end point for user logout

Example request:
curl --request POST \
    "http://localhost/api/logout" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Authorization: Bearer &lt;your_token&gt;"
const url = new URL(
    "http://localhost/api/logout"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer &lt;your_token&gt;",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());

Example response (200, logout success):


{
    "message": "User successfully logged out."
}
